# Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard development, and business reporting. The objective is to transform raw data into meaningful insights that support data-driven decision-making.

## Dataset

The dataset contains structured business data used for analysis. It includes multiple attributes related to customer and business operations, making it suitable for data cleaning, SQL querying, visualization, and reporting.

## Tools & Technologies

* **Python**

  * Pandas
  * NumPy
  * Matplotlib
  * Seaborn
* **MySQL Server**
* **Power BI**
* **Gamma** (Presentation)
* **Jupyter Notebook**

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python using Pandas.
* Performed an initial inspection of the data.

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions and trends.
* Identified missing values, duplicates, and outliers.
* Created visualizations to better understand the dataset.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected data types where required.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned data into MySQL Server.
* Wrote SQL queries to:

  * Filter records
  * Aggregate data
  * Group results
  * Sort data
  * Generate business insights

### 5. Power BI Dashboard

Developed an interactive dashboard featuring:

* Key Performance Indicators (KPIs)
* Charts and graphs
* Filters and slicers
* Business insights
* Data summary

### 6. Report Creation

Prepared a report summarizing:

* Data cleaning process
* Key findings
* Business insights
* Dashboard observations
* Final conclusions

### 7. Presentation

Created a professional project presentation using Gamma to showcase:

* Project objective
* Methodology
* Dashboard screenshots
* Insights
* Recommendations

## Dashboard

## Results

* Improved data quality through cleaning and preprocessing.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook.
4. Run the notebook cells in sequence.
5. Import the cleaned dataset into MySQL Server and execute the SQL queries.
6. Open the Power BI (.pbix) file to explore the dashboard.
7. View the report and Gamma presentation for project documentation.
